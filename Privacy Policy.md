# 隐私政策（Privacy Policy）

生效日期：【2026-01-02】
应用名称：【YMC150】
联系人邮箱：【myeongchungyang@gmail.com】

本隐私政策说明我们如何在使用 Threads API 的过程中收集、使用、存储与保护数据。本应用主要用于学术研究。

## 1. 我们收集与处理的数据类型
我们可能处理以下类型的数据（以 Threads 平台公开数据与 API 返回数据为限）：
- 公开帖子内容与元数据（文本、发布时间、公开链接等）
- 公开互动指标（点赞数、回复数等）
- 公开账号信息（用户名、公开ID、公开简介等，视 API 返回范围而定）
- 技术日志（请求时间、错误日志、速率限制信息等，用于运维与安全）

我们不主动收集敏感个人信息（如身份证号、支付信息等）。

## 2. 数据使用目的
我们仅在以下目的下使用数据：
- 学术研究与统计分析（例如：传播网络分析、话题分析、情绪分析等）
- 改进研究流程的可靠性与可复现性
- 安全与防滥用（例如：检测异常请求、保护系统稳定性）

## 3. 数据共享与披露
除以下情形外，我们不会向第三方出售或出租数据：
- 法律法规要求或监管机构合法请求
- 学术发表所需的聚合统计或匿名化结果（不包含可识别个人身份的信息）
- 与研究合作方共享时：仅共享必要的最小数据，并尽可能进行匿名化/去标识化

## 4. 数据存储、保留与安全
- 存储地点：本地加密硬盘】
- 保留期限：项目结束后 12 个月
- 安全措施：访问控制、最小权限、加密存储（如适用）、日志审计等。

## 5. 你的权利
你可以请求：
- 访问与更正（若我们持有可定位到你的数据）
- 删除（详见“数据删除说明”页面）
- 限制处理（在可行范围内）

## 6. 未成年人
本应用不以未成年人为目标用户。若你认为我们处理了未成年人相关的可识别数据，请联系邮箱以便我们评估与处理。

## 7. 政策更新
我们可能不定期更新本政策。更新将通过本页面发布，并在顶部更新生效日期。
