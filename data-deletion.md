# 数据删除说明（Data Deletion Instructions）

生效日期：【2026-01-02】
应用名称：【YMC150】
联系方式：【myeongchungyang@gmail.com】

## 1. 我们处理哪些数据
本应用通过 Threads API 访问并处理与研究目的相关的数据。通常包括：公开可见的帖子内容、发布时间、公开互动指标（如点赞/回复数）、公开账号标识信息（如用户名/公开ID）等。

## 2. 数据存储与保留
- 若我们将数据保存到研究数据库：仅用于学术研究与统计分析，采取访问控制与最小化保存原则。
- 数据默认保留期限为：项目结束后 12 个月；到期后将进行匿名化处理或删除。

## 3. 如何请求删除
如果你希望删除与您相关的应用处理数据，请按以下方式提交请求：
- 发送邮件至：【myeongchungyang@gmail.com】
- 邮件标题：Data Deletion Request – 【YMC150】
- 邮件内容请包含：
  1) 你的 Threads 用户名（@xxx）  
  2) 你希望删除的范围（全部/某时间段/某些链接或帖子）  
  3) 便于我们定位的数据线索（如帖子链接、时间范围等）

## 4. 处理时限与确认
我们将在收到请求后【30】天内完成处理，并通过邮件确认删除结果或说明无法删除的原因（例如法律合规要求或数据已不可逆匿名化）。

## 5. 例外情形
若相关数据已被不可逆匿名化并用于聚合统计，或出于法律合规/审计要求需要保留最小记录，我们可能无法删除全部信息，但会尽最大努力删除可识别个人的部分。
